android.permission.ACCESS_ADSERVICES_CUSTOM_AUDIENCE
Permite que una aplicación acceda a los datos de atribución proporcionados por la API AdServices de Android.

android.permission.ACCESS_ADSERVICES_TOPICS
Permite que una aplicación acceda a las categorías de intereses del usuario (también llamadas "temas") según lo determinado por la API AdServices de Android.
